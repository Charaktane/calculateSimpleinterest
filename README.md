# Calculate Simple Interest

Aplicação desenvolvida em Java para calcular juros simples com base em valores informados pelo usuário.

## Funcionalidades
- Cálculo de juros simples a partir do capital, taxa e tempo
- Exibição do valor final com juros

## Tecnologias
- Java

## Como usar
1. Clone o repositório
2. Abra no NetBeans ou VSCode
3. Execute a classe principal e insira os valores solicitados
